# Authorization-RegistrationFormPHP
 Форма Авторизации/Регистрации на PHP
 
 Реализована форма входа на страницу и регистрацию нового пользователя.
 
 Таблица БД содержит:
 
  •	Логин
  
  •	Пароль
  
  •	Адрес электронной почты
